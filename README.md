**RCB Auction Strategy Plan – SQL Project**:
This project analyzes IPL player performance data using SQL to design an optimized auction strategy for Royal Challengers Bangalore. The goal is to identify high-performing and cost-effective players to build a balanced team.
**Project Overview**
Designed a data-driven auction strategy for Royal Challengers Bangalore using SQL
Analyzed IPL player performance to identify high-value and consistent players
#**Problem Statement**
Teams often overspend on inconsistent players
Lack of data-driven decision-making in auctions
Need to identify undervalued players
#**Objectives**
Analyze batting and bowling performance metrics
Identify top players by role (batsman, bowler, all-rounder)
Find players with high performance and low cost
Build a balanced and cost-effective team
#**Tools & Technologies**
SQL (MySQL)
Excel (if used for preprocessing)
#Key SQL Concepts Used
GROUP BY for aggregation
JOIN to combine datasets
WHERE for filtering conditions
Aggregate functions (SUM, AVG)
Window functions (RANK, DENSE_RANK)
**#Key Insights**
Identified consistent top-performing players across seasons
Found undervalued players with strong performance metrics
Highlighted high-cost players with low consistency
Emphasized the importance of all-rounders
#Conclusion
Recommended a balanced team with a mix of star and undervalued players
Suggested focusing on consistency over one-season performance
Optimized budget allocation across roles
